# Foundations of Computer Science Coursework

This repository contains my coursework for the **Foundations of Computer Science** module.  
The project demonstrates key database and computing concepts including **data normalization, relational database design, and SQL implementation using MySQL with Docker**.


# Tasks Overview

## Task 1 

This task is about:

- Encoding formats: ASCII, Hex, Base64, URL Encoding
- HTTPS transmission of encoded data
- Encoding-based attacks and filter bypass testing

## Task 2
Analysis of data issues including:

- Data redundancy
- Update anomalies
- Insertion anomalies
- Deletion anomalies

## Task 3
Database normalization and SQL implementation.

This includes:

- Unnormalized table
- First Normal Form (1NF)
- Second Normal Form (2NF)
- Third Normal Form (3NF)
- ER diagram
- SQL table creation
- Join queries

The database implementation was performed using **MySQL running inside Docker containers**.
